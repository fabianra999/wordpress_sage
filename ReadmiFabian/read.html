Wordpress inicio de tema con sage

Quitarle a las etiquetas php "|- -|" al final y al princpio y poner mayorq menorq

  • descargar tema sage de https://roots.io/
  • descargar plugin magic-fields-2 plugin
  • configurar wp-config base de datos y agregar al final
    //config tema sage
    define('WP_ENV', 'development');
  • crear tp homo o el que se quiera con un comentario que nos permite escoger en wordpress la pagina
    php |-?php /* Template Name: Home */ ?-|
  • importar tpl cuado por ejemplo el home se divide en nosotros, temas y se quiere traer al home
    |-?php get_template_part('home/nosotros'); -?|
    |-?php get_template_part('home/temas'); -?|
  • para llamar post types de magic field el markup
    |-?php query_posts('post_type=beneficios');
    while (have_posts()) : the_post(); ?-|


    |-?php endwhile; ?-|
  • Traer campo de magic field dentro de post types
    |-?php echo get('quienes_somos_contenido'); ?-|
  • traer grupo duplicable con magic field :donde ('listado_beneficios_beneficio'); es el nombre del grupo

    |-?php
    $beneficios = get_field('listado_beneficios_beneficio');
    foreach($beneficios as $key => $beneficio) :
    $claseB = ($key % 2 == 0) ? 'odd' : 'even';
    ?-|
    |-div class="col-xs-12 col-sm-5 col-sm-offset-1 alingTop col-centered |-?php echo $claseB; ?-| "-|
    |-p-| |-?php echo $beneficio; ?-| |-/p-|
    |-/div-|
    |-?php endforeach; ?-|
